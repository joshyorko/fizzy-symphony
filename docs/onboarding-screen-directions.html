Onboarding Screen Directions

Six intentionally different first-run screens for fizzy-symphony. Each option keeps the visible board, golden ticket, setup review, and daemon status model in front of the user.

Use this as a comparison board, not final UI. The strongest direction depends on whether onboarding should optimize for confidence, speed, operator control, or board comprehension.
